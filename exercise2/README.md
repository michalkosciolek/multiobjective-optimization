Given an undirected graph. In the graph, there is a distinguished starting node representing the home. Every other node represents a store where certain products can be purchased (the list of products available at each store must be provided in the input). The input also includes a shopping list and a maximum amount of time during which shopping may be performed. Implement an algorithm that prioritizes the products to be purchased (based on the number of products that can be bought at each store) and attempts to buy as many products as possible before the given maximum time elapses. The shopper must return to the starting node before the time limit expires. Each edge label denotes the time required to travel from one node to another. Assume that making purchases (remaining at a node) takes a fixed, constant amount of time, regardless of the node and the number of items purchased. While traversing a path, one may either enter a node (to make purchases) or skip it.

NOTE: The (arbitrary) graph must be read from input and must not be hard‑coded into the algorithm.
